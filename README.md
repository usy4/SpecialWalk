# SpecialWalk
SpecialWalk is plugin to make your walk so special with ore`s

![image](https://user-images.githubusercontent.com/68358524/210122015-d3c5f471-d51e-41dc-8bdd-0f267eb62d19.png)

# How to use
Usage: `/specialwalk <on/off/menu>` 
# Permissions
```
permissions:
  specialwalk.command:
    default: true
  specialwalk.coal:
    default: op
  specialwalk.iron:
    default: op
  specialwalk.gold:
    default: op
  specialwalk.redstone:
    default: op
  specialwalk.lapis:
    default: op
  specialwalk.diamond:
    default: op
  specialwalk.emerald:
    default: op
  specialwalk.quartz:
    default: op
```
