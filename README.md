# SpecialWalk
SpecialWalk is plugin to make your walk so special with ore`s
# How to use
Usage: `/specialwalk <on/off/menu>` 
# Permissions
```
permissions:
  specialwalk.command:
    default: true
  specialwalk.coal:
    default: op
  specialwalk.iron:
    default: op
  specialwalk.gold:
    default: op
  specialwalk.redstone:
    default: op
  specialwalk.lapis:
    default: op
  specialwalk.diamond:
    default: op
  specialwalk.emerald:
    default: op
  specialwalk.quartz:
    default: op
```
